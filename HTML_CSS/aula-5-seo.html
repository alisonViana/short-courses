SEO optimizations

For better SEO, use only one h1 title per page. H1 titles are the first thing search engine look for on your page, the most important thing, because of that, use only one per page.

Use for "title tag" Website name - Description(slogan)
Article/Post Name - Website name

Meta tags

name="description"
Full description of the page's content. This is what is shown in Google search results
name="keywords"
Keywords used on search engines
name="author"
Page's author
